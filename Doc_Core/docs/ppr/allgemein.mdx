## Allgemein

<div className="background-container">
    Im Zuge der großen Systemumstellung im November mussten wir auch die Produktionspapiere 
    wie Kastenbegleitkarte, Montagebegleitkarte usw. neu entwickeln, bzw. in das neue 
    System migrieren.
    <br/>
    Hier erfährst du alles, was du über die neuen Papiere wissen musst.
</div>



## Das bleibt gleich
<div className="background-container">
    Folgende Dokumente bleiben gleich und werden nicht verändert:
    <br/>   
    - Endprüfprotokoll
</div>


## Das fällt weg 
<div className="background-container">
    Folgende Dokumente fallen weg:
    <br/>    
    - **Teilebedarfkarte**
        - ist im MES unter "Stückliste" abgebildet.
</div>


## Das ändert sich
<div className="background-container">
    - **Kastenbegleitkarte**, **Gebinde- Statusbegleitkarte**, **Leistungsblatt**
        - Diese drei Papiere werden auf die **Auftragsbegleitkarte** zusammengelegt.
    <br/>
    - **Prüfbegleitkarte**
        - Das Feld Mat.Art (Materialart) fällt weg, da es von niemanden benötigt wird.
    <br/>
     - **Maschinenfreigabekarte** 
        - Wir lediglich an das neue Design angepasst.
    <br/>
    - **Schäl- und Schneidbeleg**
        - Wird zu *Schneidbeleg* und wird von den Infos am Zettel aufs Wesentliche gekürzt.
    <br/>
    - **Walzwerkbeleg**
        - Wird zu *Walzbeleg* und wird von den Infos am Zettel aufs Wesentliche gekürzt.
    <br/>
     - **Einlagerungsschein** 
        - Wird durch den *PV-Deklaration* ersetzt und ebenfalls aufs Wesentliche gekürzt.
    <br/>
     - **Montagefreigabekarte** 
        - Wird lediglich an das neue Design angepasst.
    <br/>
     - **Montagebegleitkarte** 
        - Wird lediglich an das neue Design angepasst.
    <br/>
     - **FV-Sequenz** 
        - Wurde auf die wesentlichen Daten gekürzt und an das neue Design angepasst.
</div>
