<div className="background-container">
    Ein Auftrag oder eine Material Family kann pausiert werden.
    Das hat den Hintergrund, dass es manchmal zu Auftragsunterbrechungen kommen kann, beispielsweise durch einen Auftrag mit höherer Priorität, 
    der auf derselben Maschine gefertigt werden muss. 
    Der aktuelle Auftrag wird pausiert, während der mit höherer Priorität gefertigt wird. 
    Damit alle beteiligten sehen können, dass der Auftrag pausiert ist und es zu keinen Unklarheiten kommt, gibt es die Möglichkeit, einen 
    Auftrag auf Pausiert zu setzen.
</div>
