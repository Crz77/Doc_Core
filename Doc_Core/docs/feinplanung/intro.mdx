# Feinplanung

## Bereiche

Die **Feinplanung** gliedert sich im MES-System in folgende Bereiche: 

- [Sequenz](../category/sequenz)
- [Übersicht](./übersicht.mdx)
- [Logbuch](./logbuch.mdx)
- [Vorschau](./vorschau.mdx)
- [Plugins](./plugins.mdx)


## Sequenz
Die Sequenz ist das Herzstück der Feinplanung. Hier erfolgt die Verwaltung der Produktionsaufträge.
In der Sequenz werden Aufträge feinverplant, können in andere Bereiche verschoben, storniert oder abgeschlossen werden.


## Übersicht
Die Übersicht zeigt anhand eines Balkendiagramms wie viele Stunden die Maschinen/Arbeitsplätze eines Bereichs verplant sind.
