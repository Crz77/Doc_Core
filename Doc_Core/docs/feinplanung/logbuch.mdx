<div className="background-container">
Das Logbuch ist eine vereinfachte Form des Schichtreports V2 für einen Bereich.
Hier können folgende Daten eines Auftrages, der in den letzten 35 Schichten gefertigt wurde, eingesehen werden: <br/><br/>

- Stückzahl
- Produktionszeit
- Verluste
- OEE-Wert
- PEE-Wert
- Stückliste 
- Arbeitsplan

<br/>

Was genau der OEE- und der PEE-Wert sind, erfährst du [hier](../fertigung/areas/reporting/schichtreport-v2.mdx#oee---overall-equipment-effectiveness).
</div>
